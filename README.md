This project analyzes and predicts S&P 500 price movements using historical market data.


Technologies Used
- Python
- pandas – data manipulation
- yfinance – fetching market data
- scikit-learn – model training (Random Forest, Linear Regression)
- matplotlib – data visualization
- Jupyter Notebook

Future Improvements
- Add more technical indicators (RSI, MACD, Bollinger Bands)
- Try deep learning models (LSTM)
- Include a live data dashboard
